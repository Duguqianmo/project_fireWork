# 赛博烟花 - 万家灯火 用户说明书

## 📖 简介

"赛博烟花 - 万家灯火" 是一款精美的跨年/春节倒计时应用，结合了赛博朋克风格的视觉效果和烟花特效，为您带来独特的节日体验。

## ✨ 主要功能

### 1. 倒计时功能
- **跨年倒计时**：自动识别公历新年（1月1日 00:00:00）
- **春节倒计时**：自动识别农历春节（正月初一）
- **自定义倒计时**：可在配置文件中设置任意目标时间
- **实时更新**：倒计时数字每秒更新，精确到秒

### 2. 烟花特效
- **自动烟花**：倒计时结束后自动播放烟花特效
- **文字烟花**：显示祝福文字，如"新年快乐"、"万事如意"等
- **多种样式**：支持多种烟花样式和颜色
- **全屏效果**：烟花文字布满整个屏幕，从四面八方汇聚

### 3. 音效系统
- **启动音效**：点击启动按钮时播放音效
- **烟花音效**：烟花爆炸时播放相应音效
- **音效控制**：可通过设置按钮开启/关闭音效

### 4. 响应式设计
- **移动端优化**：完美适配手机、平板等移动设备
- **PC端优化**：大屏幕设备上显示更清晰的字体和效果
- **自适应字体**：根据屏幕尺寸自动调整字体大小

## 🚀 使用指南

### 首次使用

1. **打开页面**
   - 在浏览器中打开 `index.html` 文件
   - 等待页面加载完成

2. **点击启动按钮**
   - 页面中央会显示"赛博烟花，启动！"按钮
   - 点击按钮后，系统会：
     - 播放启动音效
     - 显示按钮爆炸特效
     - 开始倒计时
     - 激活所有动画和音效

3. **观看倒计时**
   - 倒计时面板会显示距离目标时间的剩余时间
   - 格式：`X天 XX:XX:XX`（天数为0时自动隐藏）
   - 倒计时类型会自动识别（跨年/春节）

### 倒计时结束后

- **庆祝文字**：显示配置的祝福文字（如"恭祝全球华人新年快乐！"）
- **文字烟花**：全屏显示祝福文字，持续1分钟
- **烟花特效**：自动播放烟花动画

### 控制功能

#### 暂停/继续
- 点击右下角的暂停按钮可以暂停/继续倒计时和动画

#### 音效开关
- 点击右下角的音效按钮可以开启/关闭音效

#### 设置菜单
- 点击右下角的设置按钮可以打开设置菜单
- 可调整：
  - 烟花质量
  - 烟花类型
  - 烟花大小
  - 文字烟花开关
  - 自动发射开关
  - 天空光照效果
  - 长曝光模式
  - 隐藏控制按钮

## ⚙️ 配置说明

### 配置文件位置
配置文件位于根目录的 `config.js` 文件中。

### 可配置项

#### 1. 调试时间（debugTime）
```javascript
debugTime: null  // 或 "2025/12/31 23:59:00"
```
- **用途**：用于测试和演示，模拟特定时间点
- **格式**：`"YYYY/MM/DD HH:mm:ss"`
- **说明**：设置后会使用此时间作为当前时间，倒计时会基于此时间计算
- **示例**：设置为 `"2025/12/31 23:59:00"` 可以模拟距离跨年还有1分钟

#### 2. 倒计时目标时间（countdownTargetTime）
```javascript
countdownTargetTime: "2026/01/01 00:00:00"
```
- **用途**：设置倒计时的目标时间
- **格式**：`"YYYY/MM/DD HH:mm:ss"`
- **说明**：系统会自动识别是跨年还是春节，并显示相应的标题

#### 3. 倒计时文字（countdownText）
```javascript
countdownText: [
    "新年快乐，骐骥驰骋！",
    "梦虽遥，追则能达；愿虽艰，持则可圆。",
    "${year}年，新年新气象！"
]
```
- **用途**：倒计时结束后显示的祝福文字
- **格式**：字符串数组，每行一个
- **特殊变量**：`${year}` 会自动替换为目标年份
- **说明**：第一行会以较大字体显示，其他行正常显示

#### 4. 烟花文字（fireworkText）
```javascript
fireworkText: [
    "新年快乐",
    "2026行大运",
    "马年大吉",
    // ... 更多文字
]
```
- **用途**：倒计时结束后显示的烟花文字内容
- **格式**：字符串数组，每行一个文字
- **说明**：这些文字会以烟花特效的形式在全屏显示

## 🔧 常见问题

### Q1: 为什么没有声音？
**A:** 
- 确保已点击启动按钮（浏览器要求用户交互后才能播放声音）
- 检查音效按钮是否已开启
- 检查浏览器音量设置

### Q2: 倒计时不准确？
**A:** 
- 检查系统时间是否正确
- 如果使用了 `debugTime`，确保格式正确
- 刷新页面重新加载

### Q3: 移动端显示不清晰？
**A:** 
- 系统已自动优化移动端显示
- 如果仍有问题，可以尝试：
  - 调整浏览器缩放比例
  - 使用横屏模式
  - 更新浏览器到最新版本

### Q4: 如何修改祝福文字？
**A:** 
- 打开 `config.js` 文件
- 修改 `countdownText` 和 `fireworkText` 数组
- 保存文件并刷新页面

### Q5: 如何测试倒计时效果？
**A:** 
- 在 `config.js` 中设置 `debugTime` 为接近目标时间的时间
- 例如：目标时间是 `2026/01/01 00:00:00`，可以设置 `debugTime: "2025/12/31 23:59:00"`
- 保存并刷新页面即可看到倒计时效果

### Q6: 倒计时结束后文字烟花持续多久？
**A:** 
- 文字烟花会持续显示 **1分钟**
- 期间会不断有新的文字从四面八方汇聚到屏幕中央
