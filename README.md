# 赛博烟花 - 万家灯火 🎆

## 这是什么？

这是一个**超酷的跨年倒计时网页**！打开它，你就能看到：
- ⏰ **倒计时数字**：距离新年还有多少时间
- 🎆 **烟花特效**：倒计时结束后会放烟花
- ✨ **祝福文字**：屏幕上会显示"新年快乐"等祝福语
- 🎵 **音效**：有烟花爆炸的声音效果

**简单来说**：就是一个可以在电脑、手机、平板上打开的网页，用来倒计时跨年或春节，非常炫酷！

---

## 🚀 怎么用？

### 第一步：打开网页
1. 找到 `index.html` 这个文件
2. 双击打开（或者右键选择"用浏览器打开"）
3. 等待几秒钟，页面会自动加载

### 第二步：自动开始
- **不需要点击任何按钮**，页面加载完成后会自动开始
- 你会看到倒计时数字在屏幕上方显示
- 右下角有几个小按钮（暂停、声音、设置）

### 第三步：等待倒计时
- 倒计时会显示：`X天 XX:XX:XX`（天数、小时、分钟、秒）
- 当倒计时到**最后10秒**时，会显示大数字（10、9、8...1）
- 这些大数字会**跳动**，非常醒目

### 第四步：倒计时结束
- 倒计时到0后，会显示你设置的祝福文字
- 屏幕上会开始放烟花
- 祝福文字会以烟花的形式从四面八方出现
- 整个过程持续约1分钟

## ⚙️ 怎么设置？

### 修改倒计时目标时间

打开 `config.js` 文件，找到这段代码：

```javascript
window.configData = {
    // 调试时间：用于模拟跨年效果，格式：YYYY/MM/DD HH:mm:ss
    // 如果设置了此值，系统会使用此时间作为当前时间进行倒计时
    // 例如：设置为 "2025/12/31 23:59:00" 可以模拟距离跨年还有1分钟
    // 设置为 null 或空字符串则使用系统当前时间
    debugTime: null,

    // 倒计时目标时间
    countdownTargetTime: "2026/01/01 00:00:00",
    // ...
}
```

**修改方法**：
- `countdownTargetTime`：改成你想要倒计时的时间
  - 例如：`"2026/01/01 00:00:00"` 就是2026年1月1日零点
  - 例如：`"2025/12/25 20:00:00"` 就是2025年12月25日晚上8点
- `debugTime`：用于测试，设置后会模拟这个时间
  - 例如：设置 `"2025/12/31 23:59:30"` 可以快速看到倒计时效果
  - 设置为 `null` 或 `""` 就使用真实时间

### 修改祝福文字

在 `config.js` 文件中，找到：

```javascript
// 倒计时文本（倒计时结束后显示）
countdownText: [
    "${year} Happy New Year!"
],

// 烟花祝福文本（以烟花形式显示的文字）
fireworkText: [
    "新年快乐",
    "2026行大运",
    "马年大吉",
    // ... 可以添加更多
]
```

**修改方法**：
- `countdownText`：倒计时结束后显示的大文字
  - 可以写多行，用逗号分隔
  - `${year}` 会自动替换成目标年份
- `fireworkText`：烟花形式的祝福文字
  - 可以添加任意数量的祝福语
  - 每行一个，用逗号分隔

**示例**：
```javascript
countdownText: [
    "2026年新年快乐！",
    "祝大家身体健康，工作顺利！",
    "愿新的一年心想事成！"
],

fireworkText: [
    "新年快乐",
    "恭喜发财",
    "万事如意",
    "身体健康",
    "工作顺利",
    "学业有成"
]
```

## 🎮 控制按钮说明

页面右下角有三个小按钮：

### 1. 暂停/播放按钮
- 点击可以暂停或继续倒计时和动画
- 暂停时，倒计时会停止，烟花也不会动

### 2. 声音按钮
- 点击可以开启或关闭音效
- 开启时，烟花爆炸会有声音
- 关闭时，完全静音

### 3. 设置按钮（齿轮图标）
- 点击打开设置菜单
- 可以调整：
  - **烟花类型**：选择不同的烟花样式
  - **烟花大小**：大烟花更震撼，但可能卡顿
  - **画质**：高质量更漂亮，但需要更好的设备
  - **文字烟花**：开启/关闭文字烟花效果
  - **自动发射**：开启后会自动放烟花
  - **天空照明**：烟花爆炸时背景会变亮
  - **保留轨迹**：烟花会留下残影
  - **隐藏控制按钮**：隐藏所有按钮，适合截图
  - **全屏**：切换到全屏模式
  - **倒计时目标时间**：直接在设置里改时间
  - **倒计时文案**：修改倒计时结束后的文字
  - **烟花文字**：修改烟花形式的祝福文字

## ❓ 常见问题

### Q1: 为什么没有声音？
**A:** 
1. 检查右下角的**声音按钮**是否开启（图标应该是"声音开启"的样子）
2. 检查电脑/手机的音量是否打开
3. 有些浏览器需要用户点击后才能播放声音，刷新页面试试

### Q2: 倒计时时间不对？
**A:** 
1. 检查电脑/手机的系统时间是否正确
2. 如果设置了 `debugTime`，确保格式正确：`"2025/12/31 23:59:30"`
3. 刷新页面重新加载

### Q3: 手机上看不清楚？
**A:** 
1. 试试**横屏模式**（把手机横过来）
2. 在设置里把**画质**调高
3. 确保浏览器是最新版本

### Q4: 怎么修改祝福文字？
**A:** 
1. 打开 `config.js` 文件
2. 找到 `countdownText` 和 `fireworkText` 这两行
3. 修改成你想要的内容
4. 保存文件，刷新浏览器页面

### Q5: 怎么快速测试效果？
**A:** 
1. 打开 `config.js` 文件，找到 `debugTime`
2. 设置成接近目标时间，比如：
   - 目标时间是 `2026/01/01 00:00:00`
   - 设置 `debugTime: "2025/12/31 23:59:30"`
3. 保存并刷新页面，就能看到倒计时效果了

### Q6: 为什么很卡？
**A:** 
1. 在设置里把**画质**调低
2. 把**烟花大小**调小
3. 关闭**保留轨迹**功能
4. 关闭其他占用资源的程序

### Q7: 可以全屏吗？
**A:** 
可以！点击右下角的**设置按钮**，找到"全屏"选项，勾选即可。

### Q8: 倒计时结束后会怎样？
**A:** 
1. 显示你设置的祝福文字（从下往上滑出）
2. 开始放烟花
3. 祝福文字以烟花形式全屏显示，持续约1分钟
4. 之后可以继续观看烟花，或者刷新页面重新开始

## 📝 技术说明（可选阅读）

### 文件结构
- `index.html`：主文件，包含所有代码（CSS和JS都已内联）
- 所有功能都在一个文件里，方便分享和演示

### 主要功能
- **倒计时系统**：自动识别跨年/春节，支持自定义时间
- **3D粒子效果**：使用 Three.js 实现数字粒子效果
- **烟花系统**：Canvas 2D 渲染，支持多种烟花类型
- **响应式设计**：自动适配不同屏幕尺寸
- **音效系统**：Web Audio API 实现

### 浏览器要求
- 支持 HTML5 Canvas
- 支持 Web Audio API
- 支持 ES6+ JavaScript
- 现代浏览器（2019年后）
